## About

**Frizzy-Zebra-Feet** is a repository that collects information about various pear varieties. It is intended as an easy-to-navigate reference for growers, researchers, or anyone curious about pears.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/battlesbojte999-tech/Frizzy-Zebra-Feet.git
cd Frizzy-Zebra-Feet
```

### 2. Browse the content

All catalogue entries live at the root of the repository. Open any file in your preferred text editor or Markdown viewer.

### 3. Environment variables

This project does not currently require any environment variables or secrets. If that changes, a `.env.example` file will be added to document required keys.

## Contributing

Contributions are welcome! To add a new pear variety or improve existing entries:

1. **Fork** the repository on GitHub.
2. **Create a branch** for your change:
   ```bash
   git checkout -b feature/add-bosc-pear
   ```
3. **Make your changes** and commit with a descriptive message:
   ```bash
   git commit -m "Add Bosc pear entry with growing notes"
   ```
4. **Push** your branch:
   ```bash
   git push origin feature/add-bosc-pear
   ```
5. **Open a Pull Request** against the default branch.

Please keep entries factual and cite sources where possible.
